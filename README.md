# Chat Application (Java Multithreading + Socket Programming)

## 🚀 Overview 
This project is a real-time chat application built using Java Socket Programming and Multithreading. It allows multiple clients to communicate with each other through a server.

## 🛠 Tech Stack
- Java
- Socket Programming
- Multithreading
- Networking Concepts

## ✨ Features
- Real-time messaging
- Multiple client support
- Server-client architecture
- Thread-based communication
- Continuous message exchange

## 📂 Project Structure
src/
 ├── server/
 ├── client/
 ├── handler/
 └── main classes

## ▶ How to Run
1. Clone the repository
2. Run Server file first
3. Run multiple Client instances
4. Start chatting between clients

## 📌 Important Note
- Server handles multiple clients using threads
- Communication is socket-based
- Runs in console (no GUI)
